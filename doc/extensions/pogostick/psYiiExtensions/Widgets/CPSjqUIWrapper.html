Class CPSjqUIWrapper

The CPSjqUIWraqpper is a wrapper for any jQuery UI widget

This class must be instantiated by using the static method (@link create). If you do not use that method, the options will not initiate properly.

Here we create an accordion:

  1.  //    Create an accordian...
  2.  CPSjqUIWrapper::create( 'accordion', array( 'header' => 'h3' ) );
  3.  
  4.  ...
  5.  
  6.  <html>
  7.  
  8.  ...
  9.  
  10.  <h2 class="demoHeaders">Accordion</h2>
  11.  <div id="accordion">
  12.      <div>
  13.          <h3><a href="#">First</a></h3>
  14.          <div>Lorem ipsum dolor sit amet. Lorem ipsum dolor sit amet. Lorem ipsum dolor sit amet.</div>
  15.      </div>
  16.      <div>
  17.          <h3><a href="#">Second</a></h3>
  18.          <div>Phasellus mattis tincidunt nibh.</div>
  19.      </div>
  20.      <div>
  21.          <h3><a href="#">Third</a></h3>
  22.          <div>Nam dui erat, auctor a, dignissim quis.</div>
  23.      </div>
  24.  </div>
  25.  
  26.  ...
  27.  
  28.  </html>

In the above example, the 3rd parameter instructs the library to execute the widget's (@link CWidget::run()) method after it has been initialized. This causes the output to render. If you wish to run the widget at a later time, use the following code:

  1.  //    Create an accordian...
  2.  
  3.  $_widget = CPSjqUIWrapper::create( 'accordion', array( 'header' => 'h3' ), false, null, 'cupertino' );
  4.  
  5.  ...
  6.  
  7.  $_widget->run();
