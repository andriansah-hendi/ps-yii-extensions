Class CPSCommonBase

Class Overview
Class Overview | Method Summary | Methods Detail | Constants Summary | Constants Detail

CPSCommonBase provides methods that are common to both CComponent and CWidget.

Since these two classes do not share a common base far enough up the subclasses tree, it is more modular to have this class provide the common methods necessary between our (@link CPSWidget) and (@link CPSComponent).
